## Problem  
Using the Starwars API, build a console app that creates a short bio for a given character
            
## Example  
**Input:** "Darth Vader"  
**Output:** "Darth Vader, born 41.9BBY, has appeared in 4 films."

## Resources  
Starwars API: https://swapi.co/

## Running  
> `dotnet run "Darth Vader"`
